Fichier Index.php

Section PHP :

    declare(strict_types=1); : Active le mode de typage strict pour améliorer la gestion des types de données.
    include 'includes/autoLoader.php'; : Inclut le fichier pour le chargement automatique des classes.
    session_start(); : Démarre une session pour utiliser les variables de session.
    $message et $alert : Récupère les messages et alertes de la session, le cas échéant.
    unset : Supprime les variables de session inutilisées.

Section HTML :

    <meta charset="utf-8"> : Définit le jeu de caractères du document.
    <meta name="viewport" content="width=device-width, initial-scale=1"> : Assure une mise en page responsive sur tous les appareils.
    <title> : Titre de la page.
    <link href="css/bootstrap.min.css" rel="stylesheet"> : Inclut les styles de Bootstrap.

Section Body :

    <section class="container mt-5"> : Conteneur principal avec une marge supérieure.
    <section class="row justify-content-center"> : Ligne centrée horizontalement.
    <section class="col-md-8"> : Colonne de taille moyenne prenant 8 colonnes sur un total de 12.
    <section class="bg-dark text-light text-center p-4 rounded mb-4"> : En-tête avec fond sombre, texte clair, et bordures arrondies.
    <section class="bg-white p-4 rounded shadow-sm"> : Conteneur des formulaires avec fond blanc, ombre et bordures arrondies.
    Formulaire de connexion et d'inscription : Utilisation des classes Bootstrap pour les champs de formulaire, les boutons et les alertes.

Scripts JavaScript :

    <script src="js/bootstrap.bundle.min.js"></script> : Inclut le fichier JavaScript de Bootstrap pour les fonctionnalités interactives.
