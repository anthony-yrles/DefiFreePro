Class Db

La classe Db permet de gérer les opérations de base de données, connexion, déconnexion, exécution des requêtes et récupération des résultats.
Propriétés

    $host : L'hôte de la base de données.
    $username : Le nom d'utilisateur pour se connecter à la base de données.
    $password : Le mot de passe pour se connecter à la base de données.
    $database : Le nom de la base de données.
    $connection : La connexion MySQLi.

Méthodes

    __construct($host, $username, $password, $database) :
        Constructeur de la classe. Initialise les propriétés de connexion.
        Paramètres :
            $host (string) : L'hôte de la base de données.
            $username (string) : Le nom d'utilisateur.
            $password (string) : Le mot de passe.
            $database (string) : Le nom de la base de données.

    connect() :
        Établit une connexion à la base de données en utilisant les propriétés fournies.
        Retour : void.
        Exceptions : Arrête le script et affiche une erreur si la connexion échoue.

    disconnect() :
        Ferme la connexion à la base de données.
        Retour : void.

    query($sql) :
        Exécute une requête SQL sur la base de données.
        Paramètres :
            $sql (string) : La requête SQL à exécuter.
        Retour : mysqli_result en cas de succès, false en cas d'échec.

    fetch($result) :
        Récupère une ligne de résultat sous forme de tableau associatif.
        Paramètres :
            $result (mysqli_result) : Le résultat de la requête.
        Retour : array contenant la ligne de résultat, ou null si aucune ligne n'est disponible.
